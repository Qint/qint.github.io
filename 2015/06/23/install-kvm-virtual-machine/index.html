KVM下创建虚拟机

Posted on 2015-06-23   |  

这篇博客记录如何在kvm下创建虚拟磁盘，安装虚拟机。

创建新的虚拟机的过程分为：创建虚拟磁盘、从iso镜像安装系统和启动系统。

创建虚拟硬盘

此处使用qcow2格式的虚拟磁盘，这种格式支持虚拟磁盘的大小动态增长。

创建虚拟磁盘的命令是qume-img create，其他的格式可以再指定。此处使用如下命令创建虚拟磁盘文件：

可以看到目录下新创建的文件大小只有193K，不是创建的时候指定的10G。磁盘文件将会在使用的过程中动态增加，这样的好处是能够最大可能地节省磁盘空间。
